# TransferTycoon Scenario Builder

A simple form-to-JSON converter that takes form responses and generates standardized JSON files. **No AI, no APIs, no dependencies** - just a plain static website that works entirely in the browser.

## Run Locally

**Prerequisites:**  Node.js

**No AI, no APIs, no keys required!** This is a plain static website that works entirely in the browser. The app simply collects form data and converts it to JSON - no external services needed.

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
